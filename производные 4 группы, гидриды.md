# гидриды

## C 

алканы, алкены, алкины, алкадиены, арены

### получение
- гидролиз карбидов:
	- $Al_4C_3 + 12H_2O \rightarrow 3CH_4 + 4Al(OH)_3$
	- $CaC_2 + 2H_2O \rightarrow C_2H_2 + Ca(OH)_2$

## Si: $SiH_4$ (силан)

### получение
- гидролиз силицидов: $Mg_2Si + 4H_2O \rightarrow SiH_4 + 2Mg(OH)_2$ (H+)

### хим св-ва
- самовоспламеняется на воздухе: $SiH_4 + O_2 \rightarrow SiO_2 + 2H_2O$
- разлагается: $SiH_4 \rightarrow Si + 2H_2$ (500 $\degree$C в инертной среде)
