гидролиз соли слабого основания и слабой кислоты

гидролиз по катиону:

$A:+ + HOH \rightarrow AOH + H^+$ $K_г = \frac{[AOH][H^+]}{A^+} = \frac{[AOH][K_w]}{[A^+][OH^-]} = \frac{K_w}{K_b}$

гидролиз по аниону:

$A^- + HOH \rightarrow HA + OH^-$ $K_г = \frac{[HA][OH^-]}{A^-} = \frac{[HA][K_w]}{[A^-][H^+]} = \frac{K_w}{K_a}$

среда определяется сравнением констант гидролиза
