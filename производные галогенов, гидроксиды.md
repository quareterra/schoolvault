# гидроксиды (O-содержащие к-ты)

## ==фторноватистая кислота ($HF^{-1}O$)==:

- неустойчива при н.у.

### получение: $F_2(г) + H_2O(тв) \rightarrow HF(ж-ть) + HFO(г)$ 

## номенклатура кислот галогенов


| СО  | название    | остаток   | Cl       | Br       | I       |
| --- | ----------- | --------- | -------- | -------- | ------- |
| +1  | -новатистая | гипо- -ит | $HClO$   | $HBrO$   | $HIO$   |
| +3  | -истая      | -ит       | $HClO_2$ | -        | -       |
| +5  | -оватая     | -ат       | $HClO_3$ | $HBrO_3$ | $HIO_3$ |
| +7  | -ная        | пер- -ат  | $HClO_4$ | $HBrO_4$ | $H_5IO_6$ |

### вниз по таблице
- число связей Г=О растет $\Rightarrow$ O-H связи ослабевают $\Rightarrow$ pKa $\downarrow$
- число e- до заполненного внешнего уровня $\uparrow$ $\Rightarrow$ Ox способность $\downarrow$
- экранирование Cl $\uparrow$ $\Rightarrow$ увеличение вклада $\pi$-связи $\Rightarrow$ устойчивость $\uparrow$ $\Rightarrow$ Ox способность $\downarrow$ ($HClO_4$ не является Ox в водной р-ре)
- к-ты больее сильные Ox, чем их соли

### влево по таблице
- длина связи Г-О $\downarrow$ $\Rightarrow$ pKa $\downarrow$ 
- прочность $\uparrow$ 
- r(A) $\downarrow$ $\Rightarrow$ экранирование Г $\uparrow$ $\Rightarrow$ Ox способность $\downarrow$

$HClO_4$ самая сильная из этих кислот (pKa = -10)

из кислот в СО +7 $HBrO_4$ -- самый сильный Ox (тк у $H_5IO_6$ к.ч = 6)

Ox с-ть $HBrO_3 \approx HClO_3 > HIO_3$

## $HГO$ 

существуют только в растворах

### хим св-ва:
- HIO -- амфотерный гидроксид: $H^+ + IO^- \Leftrightarrow HIO \Leftrightarrow I^+ + OH^-$
- диспропорционируют: $2HClO \Leftrightarrow HClO_2 + HCl$, $2HBrO \Leftrightarrow HBrO_3 + 2HBr$ (HIO как HBrO) 

### получение 
- HClO: $2Cl_2 + H_2O + 3HgO \rightarrow Hg_3O_2Cl_2 \downarrow + 2HClO$

## $HГO_2$ (существует только $HClO_2$)

существует только в растворах

### хим св-ва:
- диспропорционирование: $4HClO_2 \rightarrow 2ClO_2 + HClO_3 + HCl + H_2O$

### получение 
- $Ba(CLO_2)_2 + H_2SO_4 \rightarrow 2HClO_2 + BaSO_4 \downarrow$

## $HГO_3$

$HClO_3$, $HBrO_3$ -- существуют только в растворах (30%, 50% соответственно)

$HIO_3$ -- б/ц кристаллы

### хим св-ва
 термическое разложение:
- $3HClO_3 \rightarrow HClO_4 + 2ClO_2 + H_2O$ (диспропорционирование)
- $4HBrO_3 \rightarrow 2Br_2 + 5O_2 + 2H_2O$ (полное разложение)
- $2HIO_3 \rightarrow I_2O_5 + H_2O$ (дегидратация)

взаимодействие $HIO_3$ с $H_2O_2$:
- $2HIO_3 + 5H_2O_2 \rightarrow I_2 + 5O_2 + 6H_2O$
- $I_2 + 5H_2O_2 \rightarrow 2HIO_3 + 4H_2O$

### получение
- $HClO_3$: $Ba(ClO_3)_2 + H_2SO_4 \rightarrow 2HClO_3 + BaSO_4$
- $HIO_3$: $I_2O_5 \Longleftrightarrow 2HIO_3$, $I_2 + 10HNO_3(б/в) \rightarrow 2HIO_3 + 10NO_2 + 4H_2O$

## $HГO_4$

### физ. св-ва
- $HClO_4$: б/ц ж-ть 
- $HBrO_4$: неустойчива, существует только в разбавленных р-рах
- $H_6IO_6$ (ортойодная): б/ц кр.

### хим св-ва
$HClO_4$:
- Ox только в конц. р-рах
- реагирует с акт. Ме: $2HClO_4 + Ca \rightarrow Ca(ClO_4)_2 + H_2$ (==пр.==)

$H_5IO_6$:
- не дает 5-замещенных солей в обычных условиях (в основном 2-3)
- имеет координационное число 6 $\Rightarrow$ к-ты значительно слабее остальных (pKa1 = 3.29)

### получение
- $HClO_4$: $HCl(конц.) + NaClO_4 \rightarrow HClO_4 + NaCl$
- $HBrO_4$: $NaBrO_4 + H_2SO_4 \rightarrow HBrO_4 + NaHSO_4$
- $H_5IO_6$: $Ba_3(H_2IO_6) + 6HNO_3(б/в) \rightarrow 2H_5IO_6 + 3Ba(NO_3)_2$
