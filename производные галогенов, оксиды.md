# оксиды

Cl, Br, I образуют оксиды

## ==$OF_2$ -- не оксид:==

получение: $2NaOH(разб.) + 2F_2 \rightarrow 2NaF + OF_2 + H_2O$ (0$\degree$C)

хим св-ва:
- очень сильный Ox: $OF_2 +H_2O \rightarrow O_2 + 2HF$ (t$\degree$)

## $ClO_2$

### хим св-ва:

- легко диспропорционирует: $2ClO_2 + H_2O \rightarrow HClO_2 + HClO_3$ (также в NaOH)
- сильный Ox: $Na_2O_2 + ClO_2 \rightarrow NaClO_2 + O_2$

### получение
- в лаборатории: $2KClO_3 + H_2C_2O_4 \rightarrow 2ClO_2 + 2CO_2 + K_2C_2O_4 + 2H_2O$
- в промышленности: $2NaClO_3 + SO_2 + H_2SO_4 \rightarrow 2NaHSO_4 + ClO_2$

## $I_2O_5$

### хим св-ва
- ангидрид йодноватой к-ты $\Rightarrow$ р-рим
- окислитель: $I_2O_5 + 5CO \rightarrow I_2 + 5CO2$
- гидроскопичен (ступенчато присоединяет воду): $3I_2O_5 \rightarrow 2HI_3O_2 \rightarrow 6HIO_3$
