# <span style="color:#de6a5b">ПРИМЕНЕНИЕ</span>

Sc: добавляют к Al, Mg, Cr сплавам для увеличения прочности, к Cr + Sc сплавам для повышения уст. к окислению, Al + Sc сплавы обладают повыщенной электропроводностью; используется в авиастроении тк низкая плотность; оксид используется в лазерах; соединения используются как интенсивный источник света; Sc не токсичен

Y: добаляют к Ni-Cr сплавам для повыщения жароустойчивости, к стали для улучшения механических характеристик

La: карбонат используют в медицине; соединения в осветительных приборах; оксид при изготовлении опт. стекол с высокой щелочной устойчивостью и улучшенными опт. св-вами; соли используют как катализаторы при нефтепереработке; La-138 используют при лечении раковых заболеваний

Ac: источник нейтронов; маркировка белков в био. исследованиях
