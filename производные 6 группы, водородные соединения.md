# $H_2X$


## физ. св-ва

|               | $H_2O$ | $H_2S$ | $H_2Se$ | $H_2Te$ |
| ------------- | ------ | ------ | ------- | ------- |
| $Т_{пл}$      | 0      | -85    | -65     | -51     |
| $Т_{кип}$     | 100    | -60    | -41     | -4      |
| $\angle$H-Э-H | 104,5  | 92     | 91      | 90      |
| агр. сост.    | ж-ть   | газ    | газ     | газ     |
| $pKa_1$       | 14     | 7,05   | 4,0     | 3,0        |

У $H_2O$ высокая температура плавления и кипения засчет водородных связей

у $H_2O$ $sp^3$ гибридизация, у остальных почти не происходит гибридизации, тк энергия высших орбиталей у атомов сильно различаются

$H_2Se$ горит на воздухе

$H_2Te$ разлагается при 0$\degree$C

св-ва вниз по группе:
- r(А) $\uparrow$ $\Rightarrow$ длина H-X $\uparrow$ $\Rightarrow$ pKa $\downarrow$

## получение
- $H_2Te$, $H_2Se$ не получают прямой реакцией, тк требуется высокая t$\degree$
- получают из солей (сульфидов/селенидов/теллуридов)
- $Al_2S_3 + 6H_2O \rightarrow 2Al(OH)_3 + 3H_2S$ (==пр.==) (аналогично $H_2Te$, $H_2Se$)
- $FeS + H_2SO_{4(разб.)} \rightarrow 2Fe(OH)_2 + H_2S$ (==пр.==) (аналогично $H_2Te$, $H_2Se$)

## хим св-ва

### образуют соли
- $H_2S + CuSO_4 \rightarrow CuS \downarrow H_2SO_4$ (==пр.==) ($Ag_2S$, CuS, PbS, HgS нерастворимы в кислотах) 
- $NaOH + H_2S \rightarrow NaHS + H_2O$ (NaOH в недостатке)
- $2NaOH + H_2S \rightarrow Na_2S +2H_2O$ (NaOH в избытке)

### окисление
- $2H_2Se + O_{2(возд.)} \rightarrow 2H_2O + 2Se$ (аналогично $H_2Te$, $H_2S$ окисляется кислородом, но не на воздухе)
- $H_2S + I_2 \rightarrow 2HI + S$ (в р-ре)
- с сильными окислителями в нейтр. среде переходит в $X^{+6}$: $H_2X + 2H_2O + 8KMnO_4 \rightarrow K_2XO_4 + KOH + MnO_2$
- в кислой среде $H_2S$ переходит в $S$: $H_2S + H_2SO_4 + KMnO_4 \rightarrow S + MnSO_4 + K_2SO_4 + H_2O$

## другие гидриды

$H_2O_2$ -- пероксид водородв
$H_2S_2$, $H_2S_3$, $H_2S_5$, $H_2S_6$ -- полисульфидные кислоты/полисульфаны

$Na_2S_{n(р.)} + 2HCl_{(р.)} \rightarrow 2NaCl + H_2S_{(р.)}$ (n = 4-6, получается смесь)
