## оксиды N

### физ св-ва
- $N_2O$: б/ц газ, $\Delta H^0_{298}$ > 0
- $NO$: б/ц газ, $\Delta H^0_{298}$ > 0
- $N_2O_3$: синяя жидкость, устойчив при t < 0$\degree$C, $\Delta H^0_{298}$ > 0
- $NO_2$: бурый газ, $\Delta H^0_{298}$ > 0
- $N_2O_4$: б/ц летучая жидкость, $\Delta H^0_{298}$ < 0
- $N_2O_5$: б/ц тв. в-во, $\Delta H^0_{298}$ < 0

### $N_2O$ -- закись азота (веселящий газ)

получение
- разложение нитрата аммония: $NH_4NO_3 \rightarrow N_2O + 2H_2O$ (250$\degree$C)
- окисление гидроксиламина: $NH_2OH + HNO_3 \rightarrow N_2O + 2H_2O$

хим. св-ва
- поддерживает горение: $C + N_2O \rightarrow CO_2 + 2N_2$ (==пр.==)

### $NO$ -- моноксид азота

структура

![[Pasted image 20260217100730.png|400]] ![[Pasted image 20260217100838.png|300]]

получение
- в промышленности:
	- окисление $NH_3$: $4NH_3 + 5O_2 \rightarrow 4NO + 6H_2O$ (1000$\degree$C, кат. Rb/Pt)
- в лаборатории:
	- р. 30% $HNO_{3}$ c неакт. Ме: $3Cu + 8HNO3 \rightarrow 3Cu(NO_3)_2 + 2NO + 4H_2O$
	- Red нитрита в H+: $2KNO_2 + KI + 2H_2SO_4 \rightarrow 2K_2SO_4 + I_2 + 2NO + 2H_2O$

хим. св-ва
- не реагирует с водой, H+, OH-
- слабый Ox и Red:
	- Ox: ==$2NO + 3Sn + 8HCl \rightarrow SnCl_2 + 2[NH_3OH]Cl$==
	- Red: $2NO + K_2Cr_2O_7 + 4H_2SO_4 \rightarrow 2HNO_3 + Cr_2(SO_4)_3 + K_2SO_4 + 3H_2O$

### $N_2O_3$ -- азотистый ангидрид

резонансные структуры

![[Pasted image 20260217101956.png|500]]
(также двойные связи могут быть по одну сторону)

получение
- $NO_2 + NO \Longleftrightarrow N_2O_3$

хим св-ва:
- $N_2O_3 + H_2O \Longleftrightarrow 2HNO_2$
- разлагается при низких t

### $NO_2$, $N_2O_4$

$2NO_2 \Longleftrightarrow N_2O_4$ (прямая р.: -11$\degree$C, обратная р.: 140$\degree$C)

тк $NO_2$ имеет 17 e- димеризация выгодна

получение
- окисление Ме(средн. и $\downarrow$ акт.) $HNO_3$: $Zn + 4HNO_3 \rightarrow Zn(NO_3)_2 + 2NO_2$ (==пр.==)

хим. св-ва
- диспропорционирует в воде: $2NO_2 + H_2O \rightarrow HNO_2 + HNO_3$
- поддериживает горение: $2C + 2NO_2 \rightarrow CO_2 + N_2$

### $N_2O_5$ -- азотный ангидрид

получение
- вытеснением $P_2O_5$ из $HNO_3$: $2HNO_3 + P_2O_5 \rightarrow N_2O_5 + 2HPO_3$

хим. св-ва
- сильный Ox (взрывоопасен): $N_2O_5 + I_2 \rightarrow I_2O_5 + N_2$

## оксиды P

### $P_2O_3$

физ св-ва
- белое тв. в-во
- молекулярная структура

получение
- окисление фосфора: $4P + 3O_2 \rightarrow 2P_2O_3$ (t, нед. $O_2$)

хим св-ва
- является восстановителем
- $P_2O_3 + 3H_2O \rightarrow 2H_3PO_3$

### $P_2O_5$

физ. св-ва
- белое тв. в-во
- молекулярная структура

получение
- окисление фосфора: $4P + 5O_2 \rightarrow 2P_2O_5$ (t, изб. $O_2$)

хим св-ва
- не является восстановителем
- $P_2O_5 + 3H_2O \rightarrow 2H_3PO_4$ (сильное водоотнимающее средство)

## оксиды As, Sb, Bi

### физ. св-ва

$As_2O_3$ -- полимерное строение, амфотерный, не р-рим в воде
$As_2O_5$ -- полимерное строение, кислотный, р-рим в воде

$Sb_2O_3$ -- полимерное строение, амфотерный, не р-рим в воде
$Sb_2O_5$ -- полимерное строение, кислотный, не р-рим в воде

$Bi_2O_3$ -- полимерное строение, основный, не р-рим в воде
$Bi_2O_5$ -- полимерное строение, кислотный, не р-рим в воде

невозможно получить $Bi_2O_5$ правильного строения

для Sb и Bi характерны смешанные оксиды (+3 и +5)

### хим св-ва
- р. с щелочами (кроме $Bi_2O_3$)
- р. с кислотами (кроме $As_2O_5$)
- $As_2O_5$, $Sb_2O_5$, $Bi_2O_5$ -- сильные Ox: $Bi_2O_5$ >> $Sb_2O_5$ $\approx$ $As_2O_5$  
	- ==$Sb_2O_{5(тв.)} + 10HCl_{(конц.)} \rightarrow 2SbCl_3 + 2Cl_2 + 5H_2O$==
