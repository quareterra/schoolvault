# соли гидроксидов

## гипохлориты 

### хим св-ва
- сильные Ox: $4NaOH + 3NaClO + 2Fe(OH)_3 \rightarrow 2Na_2FeO_4 + 3NaCl + 5H_2O$ (==пр.==) 

### получение
- $2KOH + Cl_2 \rightarrow KClO + KCl + H_2O$ (20$\degree$C) 
- ==хлорная известь: $CaO + Cl_2 \rightarrow CaOCl_2 \: \:(Ca(ClO)_2 * CaCl_2)$==

## хлориты 

### хим св-ва
- диспр-ние: $5NaClO_2 + 2H_2SO_4 \rightarrow 4ClO_2 + 2Na_2SO_4 + 2H_2O + NaCl$

### получение
- $Na_2O_2 + 2ClO_2 \rightarrow NaClO_2 + O_2$

## хлораты

### хим св-ва
- $KClO_3 + I_2 \rightarrow 2KIO_3 + Cl_2$
- диспр-ние: $4KClO_3 \rightarrow KCl + 3KClO_4$ (t$\degree$)
- разложение: $2KClO_3 \rightarrow 2KCl + 3O_2$ (t$\degree$, $MnO_2$)

### получение
- $6KOH + 3Cl_2 \rightarrow KClO_3 + 5KCl + 3H_2O$ (60$\degree$C)
