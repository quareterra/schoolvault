# кислородные соединения
- $Me_2O$ -- оксиды; $Me_2O_2$ -- пероксиды; $MeO_2$ -- надпероксиды

## получение оксидов
- $4Li + O_2 \rightarrow 2Li_2O$ (только для Li)
- $Na_2O_2 + 2Na \rightarrow 2Na_2O$
- $2NaNO_2 + 6Na \rightarrow 4Na_2O + N_2$
- из азида: $5NaN_3 + NaNO_3 \rightarrow 3Na_2O + 8N_2$
- $2NaOH + 2Na \rightarrow Na_2O + H_2$
- $KO_2 + 3K \rightarrow 2K_2O$ (также Rb и Cs)

## хим св-ва

### р. с водой
- $Na_2O + H_2O \rightarrow 2NaOH$
- $Na_2O_2 + H_2O \rightarrow 2NaOH + H_2O_2$
- $2KO_2 + 2H_2O \rightarrow 2KOH + H_2O_2 + O_2$

### пероксиды и надпероксиды Ox засчет O
- $4KO_2 + 2CO_2 \rightarrow 2K_2CO_3 + 3O_2$
- $Na_2O_2 + CO \rightarrow Na_2CO_3$
- обе реакции используют для улавливания CO и регенерации $O_2$ из $CO_2$
