# кислородосодержащие кислоты

## O-содержащие к-ты N

### $H_2N_2O_2$ -- азотноватистая к-та

структура

![[Pasted image 20260217115456.png|400]]

физ. св-ва
- очень неустойчивая
- $pKa$ = 7

получение
- $NH_2OH + HNO_2 \rightarrow H_2N_2O_2$ (низкий выход реакции)
- из солей: $Ag_2N_2O_2 + 2HCl \rightarrow H_2N_2O_2 + 2AgCl$

хим. св-ва
- разложение: $H_2N_2O_2 \rightarrow N_2O + H_2O$ (t)
- р. с щелочани образуя гипонитриты

### $HNO_2$ -- азотистая кислота

физ. св-ва
- неустойчива, существует только в разб. р-рах
- $pk_a$ = 3,4
- токсична

получение
- растворение $N_2O_3$: $N_2O_3 + H_2O \Longleftrightarrow HNO_2$

хим. св-ва
- $2HNO_2 \Longleftrightarrow N_2O_3 + H_2O$
- разложение: $3HNO_2 \rightarrow HNO_3 + 2NO + H_2O$ (t)
- р. с щелочами образуя нитраты
- Red Ox двойственность
	- Red: $7HNO_2 + 2KMnO_4 \rightarrow 2Mn(NO_3)_2 + 2KNO_3 + 3H_2O + HNO_3$
	- Ox: $2HNO_2 + 2HCl \rightarrow Cl_2 + 2NO + 2H_2O$ (t > 100$\degree$C)


### $HNO_3$ -- азотная кислота

физ. св-ва
- $pK_a$ = -1,64

получение
- в промышленности (получают 68% -- азеотроп или 60%):
	1. р. Боша-Габера
	2. $4NH_3 + 5O_2 \rightarrow 4NO + 6H_2O$ (1000$\degree$C)
	3. $NO + O_2 \rightarrow 2NO_2$ (+$N_2O_4$)
	4. $2NO_2 + H_2O \rightarrow HNO_2 + HNO_3$
	5. при $\uparrow$ с(р-ра): $HNO_2 \rightarrow NO + NO_2 + H_2O$
	6. $\Rightarrow$ $3NO_2 + 2H_2O \rightarrow 2HNO_3 + NO$

хим. св-ва
- разлагается: $4HNO_3 \rightarrow 4NO_2 + O_2 + 2H_2O$
- реагирует почти со всеми Ме:
 ![[Pasted image 20260217121723.png]]

- с неМе часто реагирует только б/в к-та

### нитраты

хим. св-ва:
- разложение:
	- Ме акт. Mg (кроме Li): $MeNO_3 \rightarrow MeNO_2 + O_2$
	- от Mg до Cu (+Li): $MeNO_3 \rightarrow MeO + NO_2 + O_2$
	- после Cu: $MeNO_3 \rightarrow Me + NO_2 + O_2$
	- ==Fe: $4Fe(NO_3)_2 \rightarrow 2Fe_2O_3 + 8NO_2 + O_2$==
	- ==Mn: $Mn(NO_3)_3 \rightarrow MnO_2 + 2NO_2$==
- Ox в расплаве или H+ среде:
	- $MnO_{2(тв)} + 2KOH_{(ж)} + KNO_{3(ж)} \rightarrow K_2MnO_4 + KNO_2 + H_2O$ (==пр.==)

### ОВ способность O-содержащий к-т и солей N
- в + СО Ox св-ва выше в H+
- в - СО Red св-ва выше в OH-
- в H+ $N^{+/-x}$ диспропорционирует с обр. $N_2$, $HNO_3$, $NH_4^-$
- ==в OH- NO и $N_2O_4$ сопропорционируют: $2NO + N_2O_4 + 4OH^- \rightarrow 4NO_2^- + 2H_2O$==

## O-содержащие к-ты P

$H_3PO_2$ -- фосфорноватистая (гипофосфиты)
$H_3PO_3$ -- фосфористая (фосфиты)
$H_4P_2O_6$ -- фосфорноватая (фосфонаты)
$H_3PO_4$ -- фосфорная (фосфаты)
$H_4P_2O_7$ -- пирофосфорная (пирофосфаты)
$(H_3PO_4)_n$ (n = 3 или 4) -- метафосфорная (метафосфаты/полифосфаты)
$H_5P_3O_10$ -- трифосфорная (трифосфаты)

### кислоты с 1 P

$H_3PO_2$, $H_3PO_3$, $H_3PO_4$

св-ва влево:
- число P-H $\uparrow$, число OH $\downarrow$ $\Rightarrow$ $pK_a \downarrow$ $\Rightarrow$ сила к-ты $\uparrow$

### кислоты с 2 P

$H_4P_2O_6$, $H_4P_2O_7$

св-ва
- четырех основные, образуют 2-х и 4-х замещенные соли

### $H_3PO_2$

одноосновная к-та

структура

![[Pasted image 20260217131725.png|400]]

получение
- из солей: $Ba(H_2PO_2)_2 + H_2SO_4 \rightarrow 2H_3PO_2 + BaSO_4$

хим. св-ва: 
- диспропорционирует при t: $2H_3PO_2 \rightarrow H_3PO_4 + PH_3$
- Red (соли тоже):
	- $H_3PO_2 + 4FeCl_3 + 2H_2O \rightarrow H_3PO_4 + 4FeCl_2 + 4HCl$ (==пр.==)
	- $NaH_2PO_2 + 4AgNO_3 + 2H_2O \rightarrow 4Ag + H_2PO_4 + NaNO_3 + 3HNO_3$ (==пр.==)

### $H_3PO_3$

двухосновная к-та

структура

![[Pasted image 20260217132243.png|400]]

получение
- растворение ангидрида: $P_2O_3 + 3H_2O \rightarrow 2H_3PO_3$

хим. св-ва
- диспропорционирует при t: $4H_3PO_3 \rightarrow 3H_3PO_4 + PH_3$
- Red (соли тоже) слабее $H_3PO_2$Н
	- $2AgNO_3 + Na_2HPO_3 + H_2O \rightarrow 2Ag + H_3PO_4 + 2NaNO_3$ (t, без нагрева идет РИО)

### $H_4P_2O_6$

физ. св-ва
- растворима в воде, соли не растворимы

получение
- $2P_{(кр)} + 4NaClO + 2NaOH \rightarrow Na_2H_2P_2O_6 + 4NaCl$

хим. св-ва
- диспропорционирует при 25$\degree$C: $H_4P_2O_6 + H_2O \rightarrow H_3PO_4 + H_3PO_3$

### $H_4P_2O_7$ и $(HPO_3)_n$

получение: 
1. $2H_3PO_4 \rightarrow H_4P_2O_7 + H_2O$ (t = 250$\degree$C)
2. $H_4P_2O_7 \rightarrow 2HPO_3 + H_2O$ (t = 400$\degree$C)

хим. св-ва
- образует термически устойчивую соль с Ag: $Na_4P_2O_7 + 4AgNO_3 \rightarrow Ag_4P_2O_7 + 4NaNO_3$

### $H_3PO_4$

хим. св-ва 
- не разлагается
- не Ox (соли тоже)

соли
- дигидрофосфаты: растворимы все
- гидрофосфаты и фосфаты: растворимы соли щ. Ме (кроме Li)
- могут раствоятся в сильных кислотах

==качественная реакция на фосфат анион: $7H_3PO_4 + 12(NH_4)Mo_7O_{24} + 51HNO_3 \rightarrow 7(NH_4)_3(PMo_{12}O_{40}) * 3H_2O + 15H_2O + 51NH_4NO_3$==

## гидроксиды As, Sb, Bi

$H_3AsO_3$ -- мышьяковистая к-та, $pK_a$ = 9,2
$H_3AsO_4$ -- мышьяковая к-та, $pK_a$ = 2,25
$Sb(OH)_3$ -- гидроксид сурьмы (III), амфотерный
$H_3SbO_4$ -- сурьмяная к-та, $pK_a$ = 4,39
$Bi(OH)_3$ -- гидроксид висмута (III), основный

$H_3AsO_4$ и $H_3SbO_4$ -- Ox средней силы

### получение
- только $H_3AsO_4$ можно получить из оксида: $As_2O_5 + 3H_2O \rightarrow 2H_3AsO_4$
- $2SbCl_3 + 3NaCO_3 + 3H_2O \rightarrow 2Sb(OH)_3 + 6NaCl + 3CO_3$
- $Bi(NO_3)_3 + 3NaOH \rightarrow Bi(OH)_3 + 3NaNO_3$

### полимеризация $H_3AsO_4$
- $H_2AsO_4 \rightarrow H_2As3O_{10} \rightarrow (HAsO_3)_n$ (t, тримышьяковая к-та, метамышьяковая к-та)

### $H_3PO_4$, $H_3AsO_4$, $H_3SbO_4$
- $r(Pn) \uparrow$ $\Rightarrow$ Pn-O ослабевает $\Rightarrow$ $pK_a \uparrow$, Ox св-ва $\uparrow$ ($H_3AsO_4 \approx H_3SbO_4$)
