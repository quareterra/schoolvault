# водородные соединения

## B (боргидриды, бораны)
- анионный ряд $B_nH_n^{2-}$ (наиболее уст. $B_12H_12^{B_6H_6^{2-}}$)
- непредельный ряд $B_nH_{n+4}$ ($B_2H_6$, $B_5H_9$)
- предельный ряд $B_nH_{n+6}$ ($B_4H_{10}$, $B_5H_{11}$)

### производные боргидридов -- бориды (существуют почти для всех Ме)
- получают прямой р. B(кр.) с Ме при t

### диборан
- $BH_3$ очень неуст. $\Rightarrow$ простейший боргидрид -- $B_2H_6$

получение:
- $MgB_2 + 2Mg + 6HCl \rightarrow  3MgCl_2 + B_2H_6$
- $2BF_3 + 6NaH \rightarrow B_2H_6 + 6NaF$
- $2NaBH_4 + H_2SO_4 \rightarrow 2H_2 + Na_2SO_4 + B_2H_6$

структура:

B: $2s^22p^1$ на всю молекулу 12e-

![[Pasted image 20260310114033.png|500]]

банановые связи (2-электронные, 3-центровые)

хим. св-ва:
- легко гидролизуется: $B_2H_6 + 6H_2O \rightarrow 2H_3BO_3 + 6H_2$
- окисляются $O_2$ из воздуха: $B_2H_6 + 3O_2 \rightarrow H_2BO_3$
- $B_2H_6 + 2CO \rightarrow 2H_3BCO$ (p,t)
- $B_2H_6 + 2LiH \rightarrow 2LiBH_4$
- $B_2H_6 + 6CH_3OH \rightarrow 2B(OCH_3)_3 + 6H_2$
- $B_2H_6 + HCl(г) \rightarrow B_2H_5Cl + H_2$
- $B_2H_6 + NH_3 \rightarrow (HBNH)_3$ (t)

### тетрабораты

получение:
- $B_2H_6 + LiH \rightarrow Li[BH_4]$ (боргидрид Li, в среде диэтилового эфира)

хим св-ва:
- гидролиз: 
	- $Li[BH_4] + H_2O \rightarrow 4H_2 + LiBO_2$
	- $Li[BH_4] + 9H_2O \rightarrow 16H_2 + Li_2B_4O_7 + 2LiOH$
- $Li[BH_4] + I_2 \rightarrow BI_3 + LiI + 2H_2$

другие тетрабораты (n(B) всегда четное): $Na[B_3H_8]$, $K[B_4H_14]$, $K[B_11H_14]$

### бориды 
- кластеры бора
- $Mg^{2+}B_2^{-1}$, $Al^{+3}B_3^{-1}$ (соединение с мет. проводимостью, 1e- с z -1)
- аналогичны тв. р-рам внедрения

## гидриды Al, Ga, In, Tl
- In и Tl не обр. уст. гидридов

### получение:
1. $4LiH + AlCl_3 \rightarrow Li[AlH_4] + 3LiCl$ (в ср. диэтилового эфира, Ga также)
2. $Li[AlH_4] + H_2SO_4 \rightarrow 2AlH_3 + 2H_2 + Li_2SO_4$

### хим. св-ва производных гидридов
- $Li[AlH_4] + 4H_2O \rightarrow LiOH + Al(OH)_3 + 3H_2$
- $Li[AlH_4] + 3ROH (кроме CH_3OH) \rightarrow Li[AlH(OR)_3] + 3H_2$
