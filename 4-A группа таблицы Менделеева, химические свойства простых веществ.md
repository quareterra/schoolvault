# хим. св-ва

## C

### горение: $C + O_2 \rightarrow CO_2$

### переход из алмаза в графит: $C_{алм} \Longleftrightarrow C_{гр}$ $\Delta G$ = -2,8 кДж/моль, процесс очень медленный, обратный процесс возможен, но не является выгодным способом получения алмаза

### окисление
- $HNO_3$: $12C_{гр} + 18HNO_3 \rightarrow C_6(COOH)_6 + NO_2 + 6H_2O$ $t \degree C$

## Si

### реагирует с $Г_2$ (без t), $O_2$, S, P, $N_2$, B (при t)
- $Si + 2F_2 \rightarrow SiF_4$
- $Si + 2Cl_2 \rightarrow SiCl_4$ (400$\degree$C)
- $Si + O_2 \rightarrow SiO_2$ (600$\degree$C)
- $Si + 2P \rightarrow SiP_2$ (600$\degree$C)

### р-рим в щелочах: $Si + 2KOH + H_2O \rightarrow K_2SiO_3 + 2H_2$

### окисляется:
- в присутствии F-: $3Si + 4HNO_3 + 18HF \rightarrow 3H_2SiF_6 + 4NO + 8H_2O$

## Ge

### реагирует с $Г_2$, $O_2$, S (при t)
-  $Ge + 2Cl_2 \rightarrow GeCl_4$
- $Ge + S \rightarrow GeS$
- $Ge + 2S \rightarrow GeS_2$

### р-рим в щелочах: $Ge + 2KOH + 4H_2O \rightarrow K_2[Ge(OH)_6] + 2H_2$

### окисляется
- $HNO_{3(конц.)}$: $Ge + 4HNO_3 \rightarrow GeO_2 + 4NO_2 +2H_2O$
- $HNO_{3(30)}$: $Ge + 8HNO_3 \rightarrow 3Ge(NO_3)_2 + 2NO + 4H_2O$

## Sn

### реагирует с $Г_2$, $O_2$, S (при t)
- $Sn + 2Cl_2 \rightarrow SnCl_4$
- $Sn + S \rightarrow SnS$
- $Sn + 2S \rightarrow SnS_2$

### р-рим к к-тах и щелочах
- $Sn + 2HCl \rightarrow 2SnCl_2 + H_2$
- $Sn + 2KOH + 4H_2O \rightarrow K_2[Sn(OH)_6] + 2H_2$

### окисляется
- $HNO_{3(конц.)}$: $Sn + 4HNO_3 \rightarrow SnO_2 + 4NO_2 +2H_2O$
- $HNO_{3(30)}$: $Sn + 8HNO_3 \rightarrow 3Sn(NO_3)_2 + 2NO + 4H_2O$

## Pb

### реагирует с $Г_2$, $O_2$, S (при t)
- $Pb + Br_2 \rightarrow PbBr_2$
- $Pb + S \rightarrow PbS$

### р-рим в к-тах
- $Pb + 2CH_3COOH \rightarrow (CH_3COO)_2Pb + H_2$ (медленно)
- с HCl р. не идет ск $PbCl_2$ не р-рим

### окисляется
- $HNO_{3(конц или 30)}$: $Pb + 8HNO_3 \rightarrow 3Pb(NO_3)_2 + 2NO + 4H_2O$
