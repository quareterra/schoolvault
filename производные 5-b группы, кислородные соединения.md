# оксиды

## СО = +5: $V_2O_5$ $Nb_2O_5$ $Ta_2O_5$

### $V_2O_5$
амфотерный:
- $V_2O_5 + 6 KOH(изб) \rightarrow 2 K_3VO_4 + 3 H_2O$
- $V_2O_5 + HNO_3 (изб) \rightarrow 2VO_2NO_3 + H_2O$

слабый окислитель: $V_2O_5 + 6HCl (конц.) \rightarrow 2VOCl_2 + Cl_2 + 3 H_2O$

восстанавливается $H_2$ при t$\degree$: $V_2O_5 + 2 H_2 \rightarrow V_2O_3 + 2 H_2O$

разлагаетс при t$\degree$ без воздуха: 
- $3 V_2O_5 \rightarrow V_6O_13 + O_2$
- $2V_6O_13 \rightarrow 12VO_2 + O_2$

ангидрид метаванадиевой кислоты: $2HVO_3 \xrightarrow{t \degree} V_2O_5 + H_2O$

### $Nb_2O_5$ и $Ta_2O_5$
растворимы только в HF за счет комплексообразования: $Nb_2O_5 + 12 HF \rightarrow H[NbF_6] + 5 H_2O$ (также $Ta_2O_5$)

ангидриды ниобиевой и танталовой кислот соответственно: $Nb_2O_5*H_2O \rightarrow Nb_2O_5 + H_2O$ (при t$\degree$)

## СО = +4

### существует только $VO_2$

получение: $V_2O_5 + H_2C_2O_4 \rightarrow 2 VO_2 + 2 CO_2 + H_2O$

амфотерный: 
- $VO_2 + H_2SO_4 \rightarrow VOSO_4 + H_2O$
- $4 VO_2 + 2KOH \rightarrow K_2V_4O_9 H_2O$

## СО = +3

### существует только $V_2O_3$

основный: $V_2O_3 + 3 H_2SO_4 \rightarrow V_2(SO_4)_3 + 3 H_2O$

## СО = +2 

### $VO$

основный: $VO + H_2SO_4 \rightarrow VSO_4 + H_2O$

### $NbO$

кластерное строение

# пероксиды

пероксиды Ta имеют сложное строение

пероксиды Nb -- желтые
примеры: $[NbO_4]^-$, $[NbO_2(O_2^{2-})]^-$
