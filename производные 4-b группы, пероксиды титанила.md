# пероксиды титанила
- окрашеные соединениния (от оранжевых до красно-коричневых)

## получение
- H+: $TiOSO_4 + H_2SO_4 + H_2O_2 \rightarrow H_2[Ti(O_2)(SO_4)_2] + H_2O$
- OH-: $H_2[Ti(O_2)(SO_4)_2] + 8 KOH + 3 H_2O_2 \rightarrow K_4[Ti(O_2)_4] + 8H_2O + K_2SO_4$

для Zr (+4) и Hf (+4) тоже существуют пероксиды
