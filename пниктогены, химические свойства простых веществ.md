# химические свойства простых веществ

## реакции с неМе

### $N_2$
только с $H_2$ и $O_2$:
- $N_2 + 3H_2 \Longleftrightarrow 2NH_3$ (p, t, кат)
- $N_2 + O_2 \rightarrow 2NO$ (эл. ток)

### P (все)
c $O_2$ и $Г_2$:
- $4P + 5O_2 \rightarrow 2P_2O_5$ (при недостатке O -- $P_2O_3$, бел. самовоспламеняется, c кр. и черн. P р. идет только при t)
- $2P + 5Cl_2 \rightarrow 2PCl_5$ (или $PCl_5$, ==пр.==)

### $P_4$
- $P_4 + 3NaOH + 3H_2O \rightarrow PH_3 + 3NaH_2PO_2$
- $P_4 + 20HNO_3{(конц.)} \rightarrow 4H_3PO_4 + 20NO_2 + 4H_2O$
- $4P_4 + 2Na \rightarrow Na_2P_16$ (в диметилформиамиде)
- $7P_4 + 12Na \rightarrow 4Na_3(P_7)$ (в диметиловом эфире ДЭП)

### As, Sb, Bi
с $O_2$ и $Г_2$:
- As окисляется кислородом до $As_2O_3$, $Г_2$ до +5
- Sb окисляется кислородом до $Sb_2O_4$(+3 и +5), $Г_2$ до +5
- Bi окисляется кислородом до $Bi_2O_3$, $Г_2$ до +3

## окисление

### P
- c $CuSO_4$: $2P + 5CuSO_4 + 8H_2O \rightarrow 5Cu + 3H_3PO_4 + 5H_2SO_4$
- с $KClO_3$: $6P + 5KClO_3 \rightarrow 3P_2O_5 + 5KCl$
- также реагирует с другими сильными окислителями

### As
- с $HNO_{3(конц.)}$: $As + 5HNO_3 \rightarrow H_3AsO_4 + 5NO_2 + H_2O$
- ==реакция Марша: $2As + 5NaClO + 6NaOH \rightarrow 2Na_3AsO_4 + 5NaCl + 3H_2O$==

### Sb
- с $HNO_{3(конц.)}$: $Sb + 5HNO_3 \rightarrow H_3SbO_4 + 5NO_2 + H_2O$

### Bi
- с $HNO_{3(конц.)}$: $Bi + 4HNO_3 \rightarrow Bi(NO_3)_3 + NO + 2H_2O$
