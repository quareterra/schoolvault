# галогениды $ЭГ_3$

|     | B                        | Al                  | Ga       | In       | Tl                                   |
| --- | ------------------------ | ------------------- | -------- | -------- | ------------------------------------ |
| F   | $BF_3$ г. наиб. уст.     | $AlF_3$             | $GaF_3$  | $InF_3$  | $TlF_3$                              |
| Cl  | $BCl_3$ г.               | $AlCl_3$            | $GaCl_3$ | $InCl_3$ | $TlCl_3$ разл. при t=153$\degree$C   |
| Br  | $BBr_3$ г.               | $AlBr_3$            | $GaBr_3$ | $InBr_3$ | $TlBr_3$ разл. при t=35-40$\degree$C |
| I   | $BI_3$ тв. разлаг. при t | $AlI_3$ светл.-кор. | $GaI_3$  | $InI_3$ светл.-ж.  | $Tl^+(I_3)^-$ полийодид, черн.                              |

все фториды (кроме B) тугоплавкие

## B
- sp2 гибридизация: $\angle = 120 \degree$

![[Pasted image 20260310124215.png|300]]

### $BF_3$: у F есть НЭП, у B свободная орбиталь, растояние B-F очень маленькое $\Rightarrow$ НЭП F и пустая орбиталь B взаимодействуют $\Rightarrow$ связь очень прочная

### получение:
- $B_2O_3 + 3CaF_2 + 3H_2SO_4 \rightarrow 2BF_3 + 3CaSO_4 + 3H_2O$ (не получают прямой р. тк очень бурная)
- $B_2O_3 + 3C + 3Cl \rightarrow 2BCl_3 + CO$ (охлаждыют и собирают ж $BCl_3$)
- $BF_3 + AlBr_3 \rightarrow BBr_3 + AlF_3$
- $Na[BH_4] + 2I_2 \rightarrow BI_3 + NaI + 2H_2$

![[Pasted image 20260310125738.png|500]]

хим св-ва:
- необратимый гидролиз:
	- $BCl_3 + 3H_2O \rightarrow H_3BO_3 + 3HCl$ ($BBr_3$, $BI_3$)
	- $4BF_3 + 3H_2O \rightarrow H_3BO_3 + 3HBF_4$ (тетрафтороборная к-та, pKa = -0,2 при $\uparrow$ c разлагается, соли -- тетрабораты, р-римы)
- р. с основаниями Льюиса: $BF_3 + NH_3 \rightarrow BF_3-NH_3$
