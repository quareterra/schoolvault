## хим св-ва $PnH_3$

### являются основаниями
- $NH_3 + H_2O \Longleftrightarrow NH_4^+ + OH_-$ ($pK_b = 4,75$)
- $PH_3$ реагирует только с сильными кислотами
- остальные почти не проявляют основных св-в

### окисляются кислородом

$NH_3$:
- $4NH_3 + 3O_2 \rightarrow 2N_2 + 6H_2O$ (без кат., t)
- $4NH_3 + 5O_2 \rightarrow 4NO + 6H_2O$ (быстро, t, кат. Rb/Pt)
- $2NH_3 + 2O_2 \rightarrow NH_4NO_3 + H_2O$ (медленно, t, кат. Rb/Pt)

$PH_3$:
- $PH_3 + O_2 \rightarrow HPO_3 + H_2O$

### РИО в жидком $NH_3$ (р-римы соли Ме, образующих комплекты, и нитраты)
- $2AgCl + Ba(NO_3)_2 \rightarrow 2AgNO_3 + BaCl_2$
- $NH_4Br + KNH_2 \rightarrow 2NH_3 + KBr$
- $Zn(NO_3)_2 + 4KNH_2 \rightarrow K_2[Zn(NH_2)_4] + 2KNO_3$ (тетраамидоцинкат калия)

## получение $PnH_3$

### $NH_3$
- гидролиз нитридов: $MgN_2 + 6H_2O \rightarrow 2NH_3 + 3Mg(OH)_2$ (кат. щелочь)
- р. щелочи с солью аммония: $2NH_4Cl_{(тв)} + Ca(OH)_{2(тв)} \rightarrow 2NH_3 + CaCl_2 + 2H_2O$
- восстановление из нитрата: $KNO_3 + 4Zn + 7KOH + 6H_2O \rightarrow NH_3 + 2K_2[Zn(OH)_4]$
- в промышленности процесс Габера-Боша: $N_2 + 3H_2 \Longleftrightarrow 2NH_3$ (==$\Delta H < 0$, $\Delta S < 0 \Rightarrow$ идет при $\downarrow t$, но очень медленно, чтобы шла быстро: p = 200 атм, t = 450$\degree$C, кат. $Fe_3O_4 + Al_2O_3$==)

### $PH_3$
- гидролиз фосфидов: $Zn_3P_2 + 6HCl \rightarrow 2PH_3 + 3ZnCl_2$
- р. соли с щелочью: $[PH_4]I + KOH \rightarrow PH_3 + KI + H_2O$
- диспропорционирование: $P_4 + 3KOH + 3H_2O \rightarrow PH_3 + 3KH_2PO_2$

### $AsH_3$, $SbH_3$
- восстановление из соли: $K_3AsO_3 + 6Zn + 9H_2SO_4 \rightarrow 2AsH_3 + 6ZnSO_4 + 3K_2SO_4 6H_2O$ ($SbH_3$ также)

## гидразин ($N_2H_4$)

### физ. св-ва
- б/ц жидкость, р-рим в $H_2O$, т/д неустойчив

### получение
- $2NH_3 + NaClO \rightarrow N_2H_4 + NaCl + H_2O$ (кат. $Mn^{2+}$)

### хим. св-ва
- основание (2 ступени диссоциации): 
	1. $N_2H_4 + H_2O \Longleftrightarrow N_2H_5^+ + OH^-$ ($pK_b1 = 6,01$, слабее $NH_3$)
	2. $N_2H_5^+ + H_2O \Longleftrightarrow N_2H_6^{2+} + OH^-$ ($pK_b2 = 14$)
- окисляется кислородом: $N_2H_4 + O_2 \rightarrow N_2 + H_2O$
- разложение: $3N_2H_4 \rightarrow N_2 + 4NH_3$
- восстановитель (соли тоже): $N_2H_5Cl + KIO_3 + HCl \rightarrow N_2 + KCl + ICl + 3H_2O$ (==пр.==)

## гидроксиламин ($NH_2OH$)

### физ. св-ва
- б/ц кр., р-рим в воде, прочные водородные связи

### получение
- ==$6[H] + HNO_3 \rightarrow NH_2OH + 2H_2O$ (эл. ток)==

### хим. св-ва
- основание: $NH_2OH + H_2O \Longleftrightarrow NH_3OH^+ + OH^-$ ($pK_b = 7,18$, слабее чем аммиак и гидразин)
- разложение: $3NH_2OH \rightarrow NH_3 + N_2 + 3H_2O$
- ОВ двойственность:
	- Red: $2NH_2OH + I_2 \rightarrow N_2 + 2HI + 2H_2O$
	- Ox: $[NH_3OH]Cl + FeCl_2 + 2HCl \rightarrow NH_4Cl + FeCl_3 + H_2O$


|         | $NH_3$ | $N_2H_4$ | $NH_2OH$ |
| ------- | ------ | -------- | -------- |
| $pK_b1$ | 4,75   | 6,01     | 7,18         |

влево по таблице:
- электронная плотность у N $\downarrow$ $\Rightarrow$ активность НЭП $\downarrow$ $\Rightarrow$ присоединение H+ $\downarrow$ $\Rightarrow$ сила основания $\downarrow$
