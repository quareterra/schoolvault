# хим св-ва

B химические инертен (особенно кристаллический)

## реакции с неМе

### B
- при н.у. только с $F_2$, с ост. при t:
	- $4B + 3O_2 \rightarrow 2B_2O_3$ (700$\degree$ C)
	- $2B + 3Cl_2 \rightarrow 2BCl_3$ (800$\degree$ C)
	- $2B + N_2 \rightarrow 2BN$ (900$\degree$ C)

### Al, Ga, In
- образуют СО +3
	- $4Al + 3O_2 \rightarrow 2Al_2O_3$ (Ga, In также)

### Tl
- $2Tl + S \rightarrow Tl_2S$
- $2Tl + 3Cl_2 \rightarrow 2TlCl_3$ (150 $\degree$ C)
	- $TlCl_3 \rightarrow TlCl + Cl_2$ (150 $\degree$ C)

## реакции с Ме

### B
- $2B + Al \rightarrow AlB_2$ (борид алюминия)

## реакции с оксидами неМе

### B
- вытесняет фосфор из оксида: $10B + 2P_2O_5 \rightarrow P_4 + 5B_2O_3$ (при t)

## реакции с водой

### B
- $2B + 3H_2O \rightarrow 3H_2 + B_2O_3$ (при очень высокой t)

### Al
- $2Al + 6H_2O \rightarrow 2Al(OH)_3 + H_2$

## реакции с кислотами

### B
- c 30% $HNO_3$: $B + 3HNO_3 \rightarrow H_3BO_3 + 3NO2$ (100 $\degree$ C)

### Al, In, Ga
- вытеснение водорода: $2Al + 6HCl \rightarrow 2AlCl_3 + 3H_2$ (In, Ga также)
- Al и Ga c $HNO_3$ пассивируются

### Tl
- $2Tl + 3CH_3COOH \rightarrow 2CH_3COOTl + H_2$

## реакции с щелочами

### B
- овр в расплаве щелочи: $2B + KClO_3 + 2KOH \rightarrow 2KBO_2 + KCl + H_2O$ (при t, могут получатся разные продукты в зависимости от c и t)

### Al, In, Ga
- окисление H+ в OH- р-ре: $2Ga + 2KOH + 10H_2O \rightarrow 2K[Ga(OH)_4(H_2O)_2] +3H_2$ (Al, In также)
