# галогениды

## получение
- метод 1: прямой р., фториды так не получают, остальные раньше получали этим методом
- метод 2: 
	- $2BeO + CCl_4 \rightarrow 2BeCl_2 + CO_2$ (800$\degree$C)
	- $2BeO + C + Cl_2 \rightarrow BeCl_2 + CO$ (900$\degree$C)
	- побочный продукт -- фосген (токсичен), $\Rightarrow$ редко используется 
- метод 3: $CaCl_2*6H_2O \rightarrow CaCl_2 + 6H_2O$ (400$\degree$C), можно получить все кроме $BeГ_2$
- фториды (кроме $BeF_2$) можно получить РИО, тк нер-римы


## физ св-ва
- фториды тугоплавкие
- к.ч.: $BeCl_2$ -- 4, $MgCl_2$, $CaCl_2$, $SrCl_2$, $BaCl_2$ -- 6

## хим св-ва
- гидролиз $BeCl_2$: 
	1. $BeCl_2 + H_2O \Longleftrightarrow [Be(H_2O)_4]^{2+}$
	2. $[Be(H_2O)_4]^{2+} \rightarrow Be(OH)_2 \downarrow + 2HCl + 2H_2O$
- $CaCl_2 + 6H_2O \rightarrow CaCl_2*6H_2O$ (сильно эндотермическая)
- $Be^{2+}$ обр. комплексы с F-: $BeCl_2 + NH_4F \rightarrow NH_4[BeF_4] + NH_4Cl$
