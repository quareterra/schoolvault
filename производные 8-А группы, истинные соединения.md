# истинные соединения

обр. в основном Xe с F и O и реже с N, Cl и C

первое открытое соединение Xe: $Xe + PtF_6 \rightarrow Xe[PtF_6]$ (позже выяснилось что истинная стр. $[XeF][Pt_2F_{11}]$)

## фториды
- получают прямой р. при разный условиях
- структура по методу Гилеспи


|      | $XeF_2$  | $XeF_4$ | $XeF_6$ |
| ---- | -------- | ------- | ------- |
| стр. | линейная | квадрат | октаэдр |

### получение
- $Xe(г) + F_2(г) \xrightarrow{Uv, 20\degree C} XeF_2$
- $Xe(г) + 3F_2(г) \xrightarrow{NiF_2, 100\degree C} XeF_6$
- $Xe(г) + 2F_2(г) \xrightarrow{NiF_2, 300\degree C} XeF_4$
- всегда с примесью $XeF_2$

### хим св-ва
 $XeF_2$ и $XeF_6$ р. с $SbF_5$ в $BrF_3$(ж):
	- $2XeF_2 + SbF_5 \rightarrow [Xe_2F_3][SbF_6]$
	- $XeF_6 + SbF_5 \rightarrow [XeF_5][SbF_6]$ (летучий)


 $XeF_2$ с $HClO_4$: $XeF_2 + HClO_4 \rightarrow [XeF]ClO_4 + HF$

 гидролизуются:
	- $2XeF_2 + 2H_2O \rightarrow 2Xe + 4HF + O_2$ (медленно при н.у.)
	- $XeF_6 + 3H_2O \rightarrow XeO_3 + 6HF$

 $XeF_4$ дисп-ет: $6XeF_4 + 12H_2O \rightarrow 2XeO_3 + 4Xe 24HF + 3O_2$
 
 используются как лабораторные Ox:
	- $S + 3XeF_2 \xrightarrow{HF(ж)} SF_6 + 3Xe$
	- $Ir + 5XeF_2 \xrightarrow{HF(ж)} 2IrF_5 + 5Xe$
	- $5XeF_2 + 2MnSO_4 + 2H_2O \rightarrow 5Xe + 2HMnO_4 + 10HF + 2H_2SO_4$
	- $RCOOH + XeF_2 \rightarrow RF + Xe + O_2 + HF$

 используется как фторирующий агент (без Ox)
	- $XeF_6 + SiO_2 \rightarrow 2XeOF_4 + SiF_4$

## оксофториды 

### получение (среда HF)
- $XeF_6 + H_2O \rightarrow XeOF_4 + 2HF$
- $XeF_4 + H_2O \rightarrow XeOF_2 + 2HF$

## фтороксенаты (достаточно уст.)

### получение
- $XeF_4 + 2CsF \rightarrow Cs_2[XeF_6]$

## оксиды

$XeO_3$ -- б/ц кр., нелетучий
$XeO_4$ -- светло-желтый газ
строение по методу Гилеспи

### получение
- $XeF_6 + 3H_2O \rightarrow XeO_3 + 6HF$
- $KXeO_6 + 4H_2SO_{4(конц.)} \rightarrow XeO_4 + 4KHSO_4 + 2H_2O$

### хим св-ва (все р. также для $XeO_4$)
- кислотные: $XeO_3 + KOH \rightarrow K[HXeO_4]$ (гидроксоксенат)
- $K[HXeO_4] + 2KOH(к) \rightarrow Xe + O_2 K_4XeO_6 + H_2O$ (пероксоксенат)
- $K_4XeO_6 + 2Na_2SO_4 + 8H_2O \rightarrow Na_4XeO_6*8H_2O \downarrow$ (белый)
- окислитель: $XeO_3 + 6MnSO_4 + 9H_2O \rightarrow 5Xe + 6HMnO_4 + 6H_2SO_4$

## кислоты
$H_2XeO_4$ -- ксенонистая, неустойчивая, соотв. $XeO_3$
$H_2XeO_6$ -- ксеноновая, средней силы, существует только в р-ре соотв. $XeO_4$

## соли $Xe^{2+}$

### получение
- $[XeF]ClO_4 + HClO_{4(конц.)} \rightarrow Xe(ClO_4)_2 + HF$
- $XeF_2 + HN(SO_2F)_2 \rightarrow FXeN(SO_2F)_2 + HF$

## гидриды 
- не образуют бинарных соединеий с H $\Rightarrow$ гидридами называют соединения со связью Xe-H (например $Xe_2OH_2$)
