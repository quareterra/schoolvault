# <span style="color:#5f9aba">ЭЛЕМЕНТЫ</span>

благородные (ранее инертные) газы: He Ne Ar Kr Xe Rn

## эл. конфигурация внешнего слоя:

He: $ns^2$ ост: $ns^2np^6$

## СО

He, Ne, Ar: 0 (нет соединений)
Kr: 0, (+2), Xe: 0, 2, 4, 6, 8 Rn: 0, 2, (+4), (+6)
действует правило четности


## св-ва вниз по группе: 
- r(A) растет $\uparrow$
- [[потенциал ионизации]] $\downarrow$ (24,6- 10,75; большой скачок He-Ne и Ne-Ar)
- [[электроотрицательность]] $\downarrow$ (5,50-2,06)

$ПИ_1$(Ar) < $ПИ_1$(F) и $ПИ_1$(Kr) $\approx$ $ПИ_1$(H) 
ЭО(Ar) < ЭО(F) и ЭО(O)
