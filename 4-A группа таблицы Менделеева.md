# <span style="color:#5f9aba">ЭЛЕМЕНТЫ</span>

C, Si, Ge, Sn, Pb

C, Si -- неметаллы; Ge -- металлоид; Sn, Pb -- металлы

## эл. конфигурация внешнего слоя: $ns^2np^2$

## СО

C, Si, Ge: -4, 0 +2, +4

Sn: 0, +2 +4

Pb: 0, +2, (+4)

## св-ва вниз по группе: 
- r(A) растет $\uparrow$
- [[потенциал ионизации]] $\downarrow$
- [[сродство к электрону]] $\downarrow$ (Si>C, тк из-за небольшого r сила отталкивания между электронами высокая)
- [[электроотрицательность]] $\downarrow$
