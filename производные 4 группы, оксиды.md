### CO (угарный газ)
устойчивое соединение; кратность связи = 3

структура

![[Pasted image 20260228143901.png|400]]

получение
- $CO_2 + C \rightarrow 2CO$
- дегидратация органических соединений

хим св-ва
- при н.у. плохо р-рим в щелочах, к-тах и воде
- при t, p, кат: $CO + NaOH \rightarrow HCOONa$

- $CO + H_2O \rightarrow CO_2 + H_2$ (t, кат)

### $CO_2$ (углекислый газ)

устойчивое соединение, кислотный оксид, при определенных t и p образует сухой лед

структура
![[Pasted image 20260228144551.png|300]]

хим св-ва
- плохо р-рим в воде: $CO_2 + H_2O \Longleftrightarrow H_2CO_3$
- р. с щелочами: $CO_2 + 2NaOH \rightarrow NaCO_3 + H_2O$
- Ox при $\uparrow$ t: $2Fe + CO_2 \rightarrow 2FeO + C$
- не поддерживает горение

### хим св-ва
- SiO и GeO распадаются при t с диспропорционированием: $2SiO \rightarrow SiO_2 + Si$
- SnO амфотерный:
	- $SnO + 2HCl \rightarrow SnCl_2 + H_2O$
	- $SnO + 2KOH + 3H_2O \rightarrow  K_2[Sn(OH)_4(H_2O)_2]$
- все +2 (кроме PbO) Red:
	- $3GeO + 12HCl + BiCl_3 \rightarrow 2Bi + 3H_2[GeCl_6] + 3H_2O$
- $PbO_2$ и $SnO_2$ Ox ($SnO_2$ очень слабый)
	- $PbO_2 + 2Mn(NO_3)_2 + 6HNO_3 \rightarrow 5Pb(NO_3)_2 + 2HMnO_4 + 2H_2O$
- $PbO_2$ распадается при t: $3PbO_2 \rightarrow Pb_3O_4 + O_2$ (280$\degree$C)

 $SiO_2$
 - хим. инертен
 - не реагирует с водой
 - реагирует со фтором $SiO_2 + 2F_2 \rightarrow SiF_4 + O_2$
 - является кислотным оксидом $SiO_2 2NaOH_({конц.}) \rightarrow Na_2SiO_3 + H_2O$ (при t, медленно)
 - восстановление при t: $SiO_2 + Mg \rightarrow MgO + SiO$ (или Si или $Mg_2Si$)
