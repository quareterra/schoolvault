# O-содержащие к-ты

## С

угольная кислота: $H_2CO_3 \Longleftrightarrow CO_2 + H_2O$

диссоциация: 
- $H_2CO_3 \Longleftrightarrow H^+ + HCO_3^-$ $pK_a$ = 3,9
- $HCO_3^- \Longleftrightarrow H^+ + CO_3^{2-}$ $pK_a$ = 10,3

### соли -- карбонаты, гидрокарбонаты
- карбонаты плохо р-римы в воде
- гидрокарбонаты хорошо р-римы в воде

## Si

### ортокремниевая к-та: $H_4SiO_4$ $pK_a$ = 9,65
- р-рима в воде

### ортокремниевая к-та: $H_2SiO_3$ $pK_a$ = 9,89
- не р-рима в воде

### соли -- силикаты (асбесты, изумруд, берилл, сподумен, тальк)
- р-римы только Li+ и Na+
- гидролизуются: $SiO_4^{4+} + H_2O \Longleftrightarrow HSiO_4^{3-} + OH$ (после 1 ступени также идет димеризация)
- ==получение стекла: $Na_2CO_3 + CaCO_3 + 6SiO_2 \rightarrow CO_2 + Na_2CaSi_6O_14$==
