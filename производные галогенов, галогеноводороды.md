# галогеноводороды

## физ. св-ва


|            | HF       | HCl  | HBr | HI  |
| ---------- | -------- | ---- | --- | --- |
| $Т_{пл}$   | -83      | -114 | -87 | -51 |
| $Т_{кип}$  | 19       | -85  | -67 | -35 |
| S          | $\infty$ | 450  | 600 | 425 |
| агр. сост. | газ      | газ  | газ | газ    |

закономерности вниз по группе:
- $Т_{пл}$ и $Т_{кип} \uparrow$, но  HF выше остальных засчет водородных связей
- $\Delta H_{обр} \uparrow$ для HF, HCl, HBr < 0, для HI > 0 

## получение

- не получают прямой реакцией (очень бурно идет и много газовых продуктов)

HF: $CaF_2 = H_2SO_4(конц.) = CaSO_4 + 2HF$
HCl: $KCl = H_2SO_4(конц.) = KHSO_4 + HCl$
HBr:  $KBr = H_3PO_4(конц.) = KH_2PO_4 + HBr$ (==другая реакция тк конц. $H_2SO_4$ окисляет Br-, то же самое с I-==)

## хим. св-ва

закономерности вниз по группе:
- кислотность $\uparrow$ засчет уменьшения E связи из за уменьшения r(A)
- Red способность $\uparrow$ (HF -- не восстановитель, HCl окисляется только в конц. р-рах, HI окисляется $Fe^{+3}$)

особенности HF: 
- межмолекулярные водородные связи
- реагирует со стеклом: $4HF + SiO_2 = SiF_4 + 2H_2O$
- ж HF -- хороший р-ль: $3HF \Longleftrightarrow H_2F^+ + HF_2^-$
