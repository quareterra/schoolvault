# гидроксиды

Ti: $TiO_2*xH_2O$ (x = 1-8) -- титановая кислота

## получение
$H_2[TiCl_6] + 6 KOH \rightarrow TiO_2*2H_2O + 6KCl + 2 H_2O$

## модификации
$\alpha-TiO_2*2H_2O \xrightarrow{-H_2O} \beta-TiO_2*H_2O \xrightarrow{t \degree} \beta-TiO_2 + H_2O$

$\alpha-TiO_2$ -- амфотерный, $\beta-TiO_2$ -- инертный

## св-ва вниз по группе
- основность $\uparrow$
- Ox способность $\downarrow$

## соли титанила ($TiO^{2+}$) и цирконила ($ZrO^{2+}$)

### получение
- $Ti(SO_4)_2 + H_2O \xrightarrow{H^+} TiOSO_4 + H_2SO_4$

### хим св-ва
- $ZrOCl_2 + H_2SO_4 \rightarrow ZrOSO_4 + 2 HCl$
- $ZrOCl_2 + 2 H_3PO_4 \rightarrow Zr(HPO_4) + 2 HCl + H_2O$
- при разбавлении $H_2O$ обр. комплексные катионы: $[Zr_4(OH)_8(H_2O)_6]^{8+} + 4SO_4^{2-}$
