# нахождение и получение

## нахождение в природе

### Zr: сфалерит (ZnS), вюртцит (ZnS), смитсонит ($ZnCO_3$)

### Cd: гринокит (CdS), отавит ($CdCO_3$)

### Hg: киноварь (HgS)

## получение

### Zn: из сульфида
- $2 ZnS + 3 O_2 \rightarrow 2 ZnO + 2 CO_2$
- $ZnO + C \rightarrow Zn + CO_2$
- очистка: $Zn \rightarrow ZnSO_4 \xrightarrow{эл. ток} Zn + H_2SO_4 + O_2$

### Cd: из отходов производства Zn 
- $Cd \rightarrow CdSO_4$
- $CdSO_4 + Zn \rightarrow ZnSO_4 + Cd$

### Hg: из сульфидов
- $HgS + O_2 \rightarrow HgSO_2$
- $4 HgS + 4 CaO \rightarrow 4 Hg + CaSO_3 + 3 CaS$
