# получение

$F_2$:
- электролиз расплава $KHF_2$ (KF:HF=1:2) тк $T_{пл}$ = 72$\degree$С
- разложение высших фторидов (t$\degree$): $2CeF_4 = 2CeF_3 + F_2$ (==пр.==)

$Cl_2$:

в лаборатории:
- окислением HCl: $2KMnO_4+16HCl = 2MnCl_2 + 5Cl_2 + 2KCl + 8H_2O$

в промышленности: 
- электролиз р-ра NaCl: $2NaCl + 2H_2O = Cl_2 + 2NaOH + H_2$


$Br_2 \: и \: I_2$: 
- вытеснением хлором из солей

$At$: 
- ядерные реакции: $^{209}_{83}Bi + ^{4}_{2}He = \: ^{211}_{88}At + 2^1_0n$
