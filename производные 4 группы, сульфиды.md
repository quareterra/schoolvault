# сульфиды

## СО +4


|            | $CS_2$                                 | $SiS_2$       | $GeS_2$  | $SnS_2$  |
| ---------- | -------------------------------------- | ------------- | -------- | -------- |
| цвет       | б/ц                                    | б/ц           | б/ц      | желтый   |
| агр. сост. | ж-ть                                   | тв. в-во      | тв. в-во | тв. в-во |
| св-ва      | токсичен<br>огнеопасен<br>хороший р-ль | гидролизуется<br>с обр. $H_2SiO_3$ |          |          |

### получение
- $CS_2$: $CH_4 + 4S \rightarrow CS_2 + 2H_2S$ (1170$\degree$C)
- $SnS_2$: $H_2[SnCl_6] + Na_2S \rightarrow SnS_2 + 6NaCl$

## СО +4

CS неустойчив

|      | GeS     | SnS        | PbS |
| ---- | ------- | ---------- | --- |
| цвет | красный | коричневый | черный    |

получение:
- SnS, GeS: $SnCl_2 + H_2S \rightarrow SnS + 2HCl$
- PbS: $(CH_3COO)_2Pb + H_2S \rightarrow PbS + 2CH_3COOH$ (или $Pb(NO_3)_2$)

### хим св-ва:
- GeS и SnS р-римы в полисульфидах
- окисление SnS и PbS:
	- $3SnS + 4HNO_3 \rightarrow 3SnO_2 + 4NO + 3S + 2H_2O$
	- $PbS + 2H_2O_2 \rightarrow PbSO_4 + 4H_2O$
