# производные кислорода

| СО   | название                     | с чем            |
| ---- | ---------------------------- | ---------------- |
| -2   | оксиды                       | со всеми Ме      |
| -1   | пероксиды                    | с активными Ме   |
| -1/2 | супероксиды <br>надпероксиды | с самыми акт. Ме |
| -1/3 | озониды                      | K, Rb, Cs                 |

## оксиды
### акт Ме: ионная связь, реагируют с $H_2O$ с образованием щелочей
### p и d Э в СО +1-+3: ковалентная связь, нерасворимы и не реагируют с $H_2O$
### p и d Э в высоких СО: молекулярная структура, р-мы, реагируют с $H_2O$ c образованием кислот

## оксиды халькогенов

### физ. св-ва
S:
- $SO_2$ -- растворим в $H_2O$, газ, легко сжимается, молекулярная структура, $sp^3$ гибридизация
- $SO_3$, реагирует с $H_2O$, с б/в $H_2SO_4$ образует олеум, ж-ть, молекулярная структура, $sp_3$ гибридизация
- также существуют $S_8O$, $S_2O$, $SO$, $S_2O_3$

Se:
- $SeO_2$ -- растворим в $H_2O$, тв. в-во, сублимирует, полимерная структура
- $SeO_3$ -- реагирует с $H_2O$, тв. в-во, полимерная структура
- также существует $Se_2O_5$

Te:
- $TeO_2$ -- плохо р-рим в $H_2O$, тв. в-во, сублимирует, полимерная структура
- $TeO_3$ -- нерастворим в $H_2O$, тв. в-во, разлагается в тв. фазе, полимерная структура

## $XO_2$

### получение

$SO_2$
- в лаборатории: $Na_2SO_3 + 2HCl \rightarrow 2NaCl + SO_2 + H_2O$ 
- в промышленньсти: $3FeS + 5O_2 \rightarrow Fe_3O_4 + SO_2$ (t$\degree$)

$SeO_2$ 
- $Se + O_2 \rightarrow SeO_2$

$TeO_2$
- $Te + 4HNO_3 \rightarrow TeO_2 + 4NO_2 + 2H_2O$

### хим. св-ва
- реагируют c щелочами образуя соли
- $SO_2$ -- типичный восстановитель
- ==$SO_2$ -- слабый окислитель в кислой среде: $SO_2 + 4HCl + 4FeCl_2 \rightarrow S + 4FeCl_3 + 2H_2O$==

## $XO_3$

### получение
- $2SO_2 + O_2 \rightarrow 2SO_2$ ($V_2O_5$, 400$\degree$C)
- $K_2SeO_4 + SO_3 \rightarrow K_2SO_4 + SeO_3$
- $H_5TeO_6 \rightarrow TeO_3 + 3H_2O$ (t$\degree$)

### хим св-ва
- $SeO_3$ и $TeO_3$ разлагаются при t$\degree$
- $SO_3$, $SeO_3$ взаимодействуют с галогенами и их производными: $SO_3 + SCl2 \rightarrow SOCl_2 + SO_2$ (==пр.==), ==$SO_2 + Cl_2 \rightarrow SO_2Cl_2$, $SO_2Cl_2 + 2NaF \rightarrow SO_2F_2 + 2NaCl$ (пр.)==

### $H_6TeO_6$
- в основном формирует 1-2-замещенные соли, иногда 3-4
- полнозам. теллураты получаются только в твердотельном синтезе при высоких t$\degree$

## $H_2XO_3$

### получение
- оксиды с водой (только $H_2SO_3$ и $H_2SeO_3$)
- $3Te + 4HNO_3 + H_2O \rightarrow 3TeO_2 * H_2O + 4NO$ (?)
- $5TeO_2 + 2KMnO_4 + 6HNO_3 + 12H_2O \rightarrow 5H_6TeO_6 + 2KNO_3 + 2Mn(NO_3)_2$

### хим св-ва
- диссоциация

### Ox-Red св-ва
- $H_2SeO_3$ - Ox, $SO_3^-$ - Red
- $H_2SeO_3 + H_2SO_3 \rightarrow Se + 2H_2SO_4 + H_2O$
- $H_2SeO_3 + 2SO_2 + H_2O \rightarrow Se + H_2SO_4$ 
- $H_2SO_3$ -- бывает только разб. $\rightarrow$ Ox св-ва слабые
- $H_2SeO_3$ -- самый сильный Ox из трех
- $H_2TeO_3$ -- плохо р-рим $\rightarrow$ Ox св-ва очень слабые

## $H_2XO_4$

### получение

$H_2SO_4$
- $SO_2 + \frac{1}{2} O_2 \Leftrightarrow SO_2$ ($V_2O_5$)
- $SO_3 + H_2O \rightarrow H_2SO_{4(неконц.)}$
- $H_2SO_4 + SO_3 \rightarrow H_2S_2O_7$ (олеум)
- $H_2S_2O_7 + H_2O \rightarrow 2H_2SO_{4}$ (88%)

$H_2SeO_4$, $H_2TeO_4$
- $H_2SeO_3 + H_2O_2 \rightarrow H_2SeO_4 + H_2O$
- $Te + 3H_2O_2 \rightarrow H_6TeO_6$ (мелко раздробленный Te)

### хим св-ва

$H_2SO_4$
- диссоциация
- $2H_2SO_4 + HNO_3 \Longleftrightarrow NO_2^+ + H_3O^- + 2H_2SO_4^-$
- Ox при $\omega$ > 70%:

сульфаты:
- часто растворимы в воде; не р-римы: $BaSO_4$, $SrSO_4$, $PbSO_4$; мало р-римы: $AgSO_4$, $CaSO_4$
- при t = 700-1000$\degree$C разлагаются (кроме щ. Ме): $MeSO_4 \rightarrow MeO + SO_2 + O_2$
- искл.: ==$Ag_2SO_4 \rightarrow 2Ag + SO_2 + O_2$; $HgSO_4 \rightarrow Hg + SO_2 + O_2$; $2FeSO_4 \rightarrow Fe_2O_3 + SO_2 + SO_3$==

$H_2SeO_4$, $H_2TeO_4$
- диссоциация
- реагируют с щелочами: $H_2SeO_4 + KOH \rightarrow K_2SeO_4 + 2H_2O$
- реагируют с гидрокарбонатами: $H_6TeO_6 + 2KHCO_3 \rightarrow K_2H_4TeO_6 + 2CO_2 + 2H_2O$ ($H_2SeO_4$ также может р. с карбонатами, а $H_6TeO_6 -- нет$)

### Ox св-ва
- $H_2SO_4$: r(S) маленький $\Rightarrow$ O полностью закрывают S $\Rightarrow$ малоктивный Ox
- $H_2SeO_4$: хороший Ox
- $H_2TeO_4$: 6 O закрывают Te $\Rightarrow$ малоактивный Ox
