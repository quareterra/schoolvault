# хим. св-ва

## c неМе

### горение в кислороде
- $S + O_2 \rightarrow SO_2$: голубое пламя
- $Se + O_2 \rightarrow SeO_2$: синее пламя
- $Te + O_2 \rightarrow TeO_2$: голубое пламя

### реакции с галогенами
- $S + 3F_2 \rightarrow SF_6$
- $2S + Cl_2 \rightarrow S_2Cl_2$
- $2S + Br_2 \rightarrow S_2Br_2$ 
- $Se + 3F \rightarrow SeF_6$
- $Se + 2Cl_2 \rightarrow SeCl_4$
- $Te + 3F_2 \rightarrow TeF_6$
- $Te + 2Cl_2 \rightarrow TeCl_4$
- $Te + 2I_2 \rightarrow TeI_4$

### $O_2$
- реагирует со всеми неМе (кроме $F_2$, $Cl_2$, благородные газы)

### S 
- реагирует со всеми неМе (кроме $I_2$, $N_2$, $Se$ и благородными газами)

### Se
- реагирует со всеми неМе (кроме $I_2$, $N_2$, S, C и благородными газами)

### Te
- реагирует только с $O_2$ и $Г_2$


## c Ме 

### $O_2$
- образует оксиды с болшинством металлов (с Ba, Na -- пероксиды, c K -- надпероксид)
### S, Se, Te
- образуют халькогениды
- $Cu + S \rightarrow CuS$ (==пр.==)
- $Fe + Se \rightarrow FeSe$ (==пр.==)
- $2K + Te \rightarrow K_2Te$ (==пр.==)

## реакция с $H_2O$ не идет

## Ox св-ва

$O_2$ -- сильный окислитель
$O_3$ -- один из сильнейших Ox:
- $O_3 + S + H_2O \rightarrow H_2SO_4$
- $O_3 + 2KI + H_2O \rightarrow I_2 + O_2 + 2KOH$ (озон в недостатке)
- ==$O_3 + 2KOH \rightarrow 2KO_3 + 5O_2 + H_2O$ (когда озон в избытке образуется взрывоопансое в-во)==

## окисление

- $S + HClO_3 + H_2O \rightarrow H_2SO_4 + HCl$
- $Te + 3H_2O_2 \rightarrow H_6TeO_6$
- $Te + 2H_2O_{(г)} \rightarrow H_2 + TeO_2$
- $Se + KBrO_3 + 2KOH \rightarrow K_2SeO_4 + KBr + H_2O$ (р-р)
- $O_2 + PtF_6 \rightarrow [O_2]^+[PtF_6]^-$

## диспропорционирование

### S
- $3S + 6KOH \rightarrow 3K_2S + K_2SO_3 + 3H_2O$ (также в $H_2O$)

### Se
- $3Se + 6KOH \rightarrow K_2SeO_3 + 2K_2Se + 3H_2O$ (также в $H_2O$) 

### Te 
- $3Te + 6KOH \rightarrow K_2TeO_3 + 2K_2Te + 3H_2O$ (также в $H_2O$) 

## сопропорционирование

### S
- $S + 2H_2SO_4 \rightarrow 3SO_2 + 2H_2O$ (t$\degree$)
- ==$S + Na_2SO_3 \rightarrow Na_2S_2O_3$ (образование тиосульфата -- не сопропорционирование)==

## образование поликатионов/анионов

- $S_8 + 3AsF_5 \rightarrow [S_8]^{2+}[AsF_6]_2^- + AsF_3$ (в ж $SO_2$)
- $4Se + K_2Se \rightarrow K_2^+[Se_5]^{2-}$
- $6Te + 6AsF_5 \rightarrow [Te_6]^{4+}[AsF_6]_4^- + 2AsF_3$ (в ж $SO_2$)
- $7Te + K_2Te \rightarrow K_2[Te_8]^{2-}$
