# хим св-ва

## р. с неМе

### с $O_2$
- Be, Mg, Ca: $2Ме + O_2 \rightarrow 2МеO$
- Sr, Ba: $Ме + O_2 \rightarrow МеO_2$ (500-700$\degree$C)


|              | Be               | Mg    | Ca          | Sr           | Ba     |
| ------------ | ---------------- | ----- | ----------- | ------------ | ------ |
| цвет пламени | коричнево-красн. | белый | красно-кор. | темно-красн. | желтый |

### c $Г_2$, S и $P_4$:
- $Ме + Cl_2 \rightarrow МеCl_2$
- $Ме + S \rightarrow МеS$
- $4Ме + 3P_4 \rightarrow 4МеP_3$ (2 $P^{-1}$ и 1 $P^0$)

### с $H_2$ (кроме Be)
- $Mg + H_2 \rightarrow MgH_2$ (p, t) 
- Ca, Sr, Ba $Ме + H_2 \rightarrow МеH_2$ (t)

## р. с водой (кроме Be)
- $Me + 2H_2O \rightarrow Me(OH)_2 + H_2$ (Mg только при t)

## р. с кислотами
- Be пассивируется с $HNO_3$ (конц)
- Mg, Ca, Sr, Ba: $Me + 2CH_3COOH \rightarrow Ca(CH_3COO)_2 + H_2$
- $4Be(OH)_2 + 6CH_3COOH \rightarrow Be_4(CH_3COO)_6O \downarrow + 7H_2O$ (оксид-гексаацетат бериллия)

## особенности Mg и Be

### реагируют с $N_2$
- $3Be + N_2 \rightarrow Be_3N_2$ (500$\degree$C)

### Mg р-рим в $NH_4Cl$, Be -- в фторидах щ. Ме и аммония
- $Mg + 2NH_4Cl + 2H_2O \rightarrow MgCl_2 + 2NH_3*H_2O + H_2$
- $Be + 4NH_4F + 2H_2O \rightarrow (NH_4)_2[BeF_4] + 2NH_3*H_2O + H_2$ (прочные комплексы с F-)

### Be р-рим в щелочах
- $Be + 2KOH + 2H_2O \rightarrow K_2[Be(OH)_4] + H_2$

## особенности щ/з Ме

### р-римы в $NH_3$(ж)
- $Me + 2NH_3 \rightarrow Sr(NH_2)_2 + H_2$
