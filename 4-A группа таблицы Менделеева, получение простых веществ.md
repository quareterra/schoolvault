# получение

## С
- добывают (углерод, графит, алмаз)

## Si 
1. добывают $SiO_2$
2. восстанавливают из оксида:
	- $SiO_2 + 2C \rightarrow 2CO + Si$
	- $SiO_2 + 2Mg \rightarrow 2MgO + Si$

## Ge
1. получают $GeO_2$ из отходов производства Zn и Ni
2. восстанавливают из оксида: $GeO_2 + 2H_2 \rightarrow Ge + 2H_2O$

## Sn
- добывают касситерит ($SnO_2$)
- восстанавливают из оксида: $SnO_2 + C \rightarrow Sn + CO_2$

## Pb
- из сульфидов:
	1. $PbS + 3O_2 \rightarrow 2PbO + 2SO_2$
	2. $PbO + CO \rightarrow Pb + CO_2$
