# кислородные соединения

## оксиды

### +2


|          | ZnO        | CdO        | HgO                |
| -------- | ---------- | ---------- | ------------------ |
|          | амфотерный | основный   | основный           |
| цвет     | белый      | коричневый | желтый или красный |
| $T_{пл}$ | 1990       | 1815       | 100                   |

ZnO -- желтеет при нагревании, тк появляются дефекты
HgO -- цепочечная структура, цвет зависит от размера частиц

$HgO + 4 KI + H_2O \rightarrow K_2[HgI_4] + 2KOH$ (тетраиодмеркурат)

### +1 (только $Hg_2O$)
разлагается: $HgO + Hg$
