# окисды

## физ св-ва

|                    | $TiO_2$                           | $ZrO_2$                                 | $HfO_2$ |
| ------------------ | --------------------------------- | --------------------------------------- | ------- |
| $Т_{пл}$           | 1870                              | 2850                                    | 2900    |
| $\Delta H^0_{обр}$ | <<0                               | <<0                                     | <<0     |
| $\Delta G^0$       | <<0                               | <<0                                     | <<0     |
| кр. модификации    | рутил, бруксит, анатаз (к.ч. = 6) | бадделит (к.ч. = 7), флюорит (к.ч. = 8) | аналогично $ZrO_2$        |

## получение

$TiO_2$: прямая реакция с $O_2$

рутил: 
1. $TiO_2 + H_2SO_4(конц.) \rightarrow TiOSO_4 + H_2O$
2. $TiOSO_4 + H_2O (пар) \rightarrow TiO_2(рутил) + H_2SO_4$ 

анатаз: 
1. $TiO_2 + 2 C + 2 Cl_2 \rightarrow TiCl_4 + 2 CO$
2. $TiCl_4 + O_2 \rightarrow TiO_2(анатаз) + 2 Cl_2$

остальные: прямой реакцией

## хим св-ва

инертны: 
- $TiO_2$ втупает в реакции при t = 1000$\degree$C
- $ZrO_2$, $HfO_2$ -- при более высоких t

$TiO_2 + CaO \rightarrow CaTiO_3$ (1200$\degree$C)
